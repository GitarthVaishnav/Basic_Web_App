README FILE FOR INTELLIGENT SCANNERS

# This application requires GPU with CUDA support #


# TO INSTALL THE DEPENDENCIES FOR THE APPLICATION #

######## Installing Basic Libraries ##############

# EITHER PERFORM THE FOLLOWING COMMANDS #
pip install django
pip install torch
pip install torchvision
pip install uuid
pip install opencv-python

# OR USE requirements.txt file #
pip install -r requirements.txt


#################################################################################

# TO RUN THE BACK-END DJANGO SERVER #

Change directory (cd) to the main folder of this application where the manage.py file is present.

In terminal type: "python manage.py runserver"

This will start the server if all the dependencies are installed perfectly. Use the url to access the application.
